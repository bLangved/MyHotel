## Getting started

### Installing

To get project up and running, use the following terminal command(s). 

1. Clone the repo:

```
git clone git@github.com:bLangved/MyHotel.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following command(s):

```
npm run dev
```
